# Boomi Component XML Reference Samples

Real component XML pulled from two Boomi sandboxes via `scripts/fetch-boomi-samples.ts`:
- **`avaxia-9FCJIF`** (initial fetch — overwritten by the larger fetch)
- **`jeracoinc-POEAF2`** (current contents — 804 samples, spread across the size spectrum)

These are the **only** schema spec the dry-run generators in `src/lib/boomi-xml.ts` are aligned against. Anything generated by Boomi Helper Suite must round-trip into a tag/attribute shape consistent with what's in this folder.

The smallest sample per type is canonical for assertions. Larger samples are kept as edge cases. Each filename embeds the size as a zero-padded prefix so `ls` sorts naturally smallest-to-largest. See `LEARNINGS.md` next to this file for the schema summary.

## Conventions captured

- Envelope: `<?xml version="1.0" encoding="UTF-8" standalone="yes"?><bns:Component xmlns:xsi=".../XMLSchema-instance" xmlns:bns="http://api.platform.boomi.com/" ...attributes...><bns:encryptedValues/><bns:description></bns:description><bns:object>...inner...</bns:object></bns:Component>`.
- Inner profile/map element declares `xmlns=""` to reset the default namespace.
- Every data element carries a sequential numeric `key` and `name`.
- Field types: `dataType="character"` (string/date/boolean) or `dataType="number"` (integer/decimal). Format details live in `<DataFormat><ProfileCharacterFormat/></DataFormat>` or `<DataFormat><ProfileNumberFormat/></DataFormat>`.
- `transform.map`'s `<Mapping>` element references profile fields via `fromKey`/`toKey` (numeric) plus `fromKeyPath`/`toKeyPath` (XPath using `*[@key='N']` segments) and a human `fromNamePath`/`toNamePath`.

## Refreshing

Run with sandbox credentials in env vars:

```bash
BOOMI_ACCOUNT_ID='avaxia-9FCJIF' \
  BOOMI_API_USERNAME='BOOMI_TOKEN.user@example.com' \
  BOOMI_API_TOKEN='your-token' \
  npx tsx scripts/fetch-boomi-samples.ts
```

The script queries ComponentMetadata for each type, fetches the 10 smallest
current components, and writes the 3 smallest to disk.
